Инструкция по заполнению списка респондентов для Национального рейтинга 2024 года

  • 1. Субъект РФ

    Укажите регион, где респондент получал услугу. (Для всех — одно значение).

  • 2. Регион местонахождения юрлица

    Укажите субъект РФ, где зарегистрировано юридическое лицо.

  • 3. Тип населенного пункта

    Выберите один из вариантов:

    • Город
    • Посёлок городского типа
    • Село
    • Деревня
    • Станица
    • Аул

  • 4. Наименование населенного пункта

    Полное название без сокращений («г.» и т. д.).

  • 5. Фамилия, Имя, Отчество

    Начинайте с заглавной буквы.

  • 6. Должность

    Укажите должность (например, Директор, Менеджер).

  • 7. Тип субъекта предпринимательства

    Выберите один из вариантов:

    • ИП
    • ООО
    • АО (ОАО, ЗАО, ПАО)
    • НКО
    • ОП

  • 8. Наименование организации

    Полное название без кавычек и типа юрлица.

  • 9. Мобильный номер телефона

    Начинается с «7», 11 цифр, без пробелов и символов (например, 79101234567).

    Несколько номеров через «;».

  • 10. Городской номер телефона

    Начинается с «7» + код города, без пробелов и символов.

    Несколько номеров через «;».

    Добавочный номер указывайте в скобках после основного (например, 74951234567(123)).

  • 11. Электронная почта

    Формат: стандартный, без пробелов и специальных символов (например, **@**, **..**, **+** и т. д.).

    Несколько адресов через «;».

Проверьте данные перед отправкой.
